## REST API
### API란
- 소프트웨어가 다른 소프트웨어로부터 지정된 형식으로부터 요청, 명령을 받을 수 있는 수단
### REST API란
- 각 요청이 어떤 동작이나 정보를 위한 것인지를 요청의 모습 자체로 추론 가능
- HTTP 요청 시 어떤 URI에 어떤 메소드를 사용할지에 대한 약속
- POST, GET, PATCH, PUT, DELETE
